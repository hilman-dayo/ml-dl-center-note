* 概念
  - 単層パーセプトロンとの共通点が多くある
  - 単層パーセプトロン
    - ステップ活性化関数
    - 入力信号の乗算された総和が
      <= 0 → 0 に変換
      >  0 → 1 に変換
  - 一般的な多層パーセプトロンと同じと考えられる
    - 双方, シグモイド活性化関数を使っている

* 活性関数
  - 非線形関数関数にしないといけない
  - 線形関数を使うと, 層を深くする意味はない
    - 隠れ層のない NN になってします
